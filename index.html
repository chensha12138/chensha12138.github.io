Hexo博客搭建记录（一）

1. 基础准备

  1. 安装node
  2. 安装github（主要命令工具）
  3. 申请github账号
  4. 安装代码编译工具
    我选的是Visual Studio Code，反正选一个工具安装，为了后面编辑config.yml文件使用。

2.安装hexo

  1. 建立文件夹
    在自己喜欢的位置新建一个文件夹作为整个博客的主文件夹，如：Blog。在该文件夹内右击鼠标，Git Bash here,打开Git
  2. 初始化hexo
>hexo init
  1. 安装插件
>npm install
  1. 启动本地服务器
>hexo s

浏览器输入地址localhost:4000即可打开博客页面，至此安装就完成。
如果浏览器一直打不开，用代码

>hexo server -p 5000

将端口4000改为5000，然后打开localhost:5000即可打开（如果你的电脑上装了福昕阅读器，没错，打不开就是它的原因！）

3.配置Git

  1. 建立Repository
    打开Github网站，Star a project。用语言编辑软件打开Bolg文件夹下_config.yml文件，翻到最下面，修改为(注意空格)：

hexo3.0版本更新之后将github.com.git改为了github.io.git。

2.部署npm

>npm install hexo-deployer-git --save
>hexo d

然后再在浏览器地址栏中输入https://username.github.io 就可以打开你的博客。

以后每次部署的步骤：

>hexo clean
>hexo g
>hexo d

4.感慨

博主没有配置ssh文件，因为网上教程很多。写这篇博客主要是因为网上有很多版本的搭建教程，但是博主遇到的问题都有没有涉及到，所以写一些东西以供参考。
