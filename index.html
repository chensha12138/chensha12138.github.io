Veröffentlicht am 2017-03-05

Hexo博客搭建记录（二）

1.主题更换

1.在官网上自己选择自己喜欢的主题，打开后按照里面的教程完成，本篇以next主题为例，主题较为简洁，适合新手选择。
2.启动Git，输入命令：

git clone https://github.com/iissnan/hexo-theme-next themes/next

等待下载完成。
3.启用next文件
打开主目录下站点配置文件_config.yml，将theme： 值改为 next。依次输入命令：

npm install hexo-renderer-jade –save
npm install hexo-renderer-sass –save

4.本地预览

hexo clean
hexo g
hexo s –debug

打开localhost:4000查看效果，满意后部署到Git上。

hexo clean
hexo g
hexo d

最后一步可能由于网络原因失败，多试几次吧（可能是博主的运气不好）。

2.写博客与上传

1.写博客
1.1新手可以用作业部落 https://www.zybuluo.com 所见即所得，写完之后直接下载，拷贝到Blog>source>_posts>blog’ name。
1.2用git

hexo n “blog’ name”

用代码编辑器即可打开Blog>source>_posts>blog’ name即可。
2.上传
打开Git

hexo clean
hexo g
hexo s

打开localhost:4000检查无误后，更新到个人的Git上去。

Hexo博客搭建记录（一）

Veröffentlicht am 2017-03-05

Hexo博客搭建记录（一）

1. 基础准备

  1. 安装node
  2. 安装github（主要命令工具）
  3. 申请github账号
  4. 安装代码编译工具
    我选的是Visual Studio Code，反正选一个工具安装，为了后面编辑config.yml文件使用。

2.安装hexo

  1. 建立文件夹
    在自己喜欢的位置新建一个文件夹作为整个博客的主文件夹，如：Blog。在该文件夹内右击鼠标，Git Bash here,打开Git
  2. 初始化hexo
>hexo init
  1. 安装插件

npm install

  1. 启动本地服务器

hexo s

浏览器输入地址localhost:4000即可打开博客页面，至此安装就完成。
如果浏览器一直打不开，用代码

hexo server -p 5000

将端口4000改为5000，然后打开localhost:5000即可打开（如果你的电脑上装了福昕阅读器，没错，打不开就是它的原因！）

3.配置Git

  1. 建立Repository
    打开Github网站，Star a project。用语言编辑软件打开Bolg文件夹下_config.yml文件，翻到最下面，修改为(注意空格)：

hexo3.0版本更新之后将github.com.git改为了github.io.git。

2.部署npm

npm install hexo-deployer-git –save
hexo d

然后再在浏览器地址栏中输入https://username.github.io 就可以打开你的博客。

以后每次部署的步骤：

hexo clean
hexo g
hexo d

4.感慨

博主没有配置ssh文件，因为网上教程很多。写这篇博客主要是因为网上有很多版本的搭建教程，但是博主遇到的问题都有没有涉及到，所以写一些东西以供参考。
